Scheduling Algorithms

Process scheduling is an activity performed by the process manager which determines which processes are in the ready state and which should be moved to the running state. This allows the operating system to allocate time intervals for processes to be executed in a manner that always keeps the CPU busy and meets system objectives such as processor efficiency and response time.
